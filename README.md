Smart-study-planner

Built a web-based productivity platform designed to help students organize their academic schedules and improve time management. Features include a weekly planner with daily scheduling, an hourly planner, Pomodoro timer, stopwatch, task manager, note-taking section, and a productivity dashboard that tracks daily progress and task completion.

Live Demo: https://studyplannerserra.netlify.app/

Features
1. Weekly Planner
Organize tasks for all seven days of the week with a structured planning system that helps users manage their academic schedule effectively.

2. Hourly Planner
Break down daily tasks hour-by-hour for better time allocation and improved productivity.

3. Pomodoro Timer
Built-in Pomodoro clock to help users stay focused using timed work sessions and short breaks.

4. Stopwatch
Allows users to track study sessions, practice durations, or productivity intervals in real time.

5.Task Manager
Create, manage, and monitor tasks efficiently with task completion tracking.

6. Notes Section
Dedicated space for writing quick notes, important reminders, and study-related information.

7. Productivity Dashboard
Displays daily productivity insights, progress tracking, and completed tasks to help users stay motivated and consistent.

8. User-Friendly Interface
Designed with a modern gold and black aesthetic to create an engaging and visually appealing study environment. Including a vision board for student motivation
